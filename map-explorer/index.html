How It Works

GenerationTiles are generated on-demand by a GPU server running a JiT flow-matching diffusion model trained on OpenStreetMap data.
Seamless stitchingUses a 9-tile overlapping scheme with hard conditioning — when zooming in, all 4 children are generated at once via 9 overlapping blocks with 50% overlap, enforced via ODE replacement.
Multi-scalez10 (root) to z17 (street level). Each zoom level quadruples the area in 4 seamless tiles.
JiT-B/16 model, trained with flow matching on 500+ cities.
Heun ODE solver, 50 steps, CFG scale 2.0.
